# Go Fullstack Boilerplate

An opinionated boilerplate for building a fullstack application with Go and Vue 3.

## Features

- [x] Webapp
  - [x] UI (Vue 3)
  - [x] Configuration
  - [x] Logging
  - [x] Embed static files
  - [x] Modular architecture
  - [ ] WebSocket
  - [ ] Email
  - [ ] Built-in Modules
    - [ ] Authentication
    - [ ] User Management
    - [ ] Role Management
    - [ ] Permission Management
    - [ ] Audit Log
    - [ ] File Storage
    - [ ] etc.
- [ ] Database
- [ ] ORM
- [ ] Migration
- [ ] Testing
- [ ] CI/CD
